# PINとは

PINとパスワードは違うもの  

端末内に格納してあるPINと、自身が入力したPINを照合する→クリアすればアクセスできる（＝ネットワーク上にPINの内容が流出しない）  


例：キャッシュカードのICカード内に登録されているPINと、自身が入力したPINを照合する  
スマートフォン内部にあるPINと、自身が入力したPINを照合する  

これらは2要素認証（ICカード自体を所有してないといけない＋PINの内容を知ってないといけない）
