
# リンクタグのURLを空欄にしたら、現在のページが表示されるような設定になっている可能性がある

```html
       <div class="item6">
         <div class="headline">ポートフォリオサイト</div>
         <a href="" target="_blank">https://hoge.github.io/myPortfolio/</a>
       </div>
       <div class="item8">Created by CSS Grid Layout.</div>
     </div>

```
