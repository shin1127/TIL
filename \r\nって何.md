# \r\nって何

WindowsOSにおける改行をあらわす

Pythonで似たようなことをやってはいたけど（= \n）、厳密にはあまり記憶していなかった。  
昔学習したような気もする。  


> \n(LF)：Unix系OS全般、Mac OS X  
> \r\n(CR+LF)：Windows系OS  
> \r(CR)：古いMac OS（9以前）
