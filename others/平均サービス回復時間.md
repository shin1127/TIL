# 平均サービス回復時間とか

- 平均故障間隔(MTBF:mean time between failures)
- 平均修復時間（MTTR：mean time to repaier)：障害発生から修復完了までの平均
- 平均サービス回復時間（MTRS:mean time to restore service)：サービス停止から回復するまでの時間
